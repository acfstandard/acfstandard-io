# acfstandard.io — ACF® Developer Documentation

Official developer documentation for the **Agentic Commerce Framework®** (ACF®),
deployed at [https://acfstandard.io](https://acfstandard.io).

Sister site to:
- [acfstandard.com](https://acfstandard.com) — the marketing vitrine
- [compliance.acfstandard.com](https://compliance.acfstandard.com) — the SaaS
- [github.com/acfstandard/acf-mcp](https://github.com/acfstandard/acf-mcp) — the MCP server

## Local dev

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Stack

- Next.js 15 (App Router)
- React 19
- Tailwind CSS 3
- Deployed on Vercel
